# NatureTrip

![](https://github.com/hirundos/NatureTrip/blob/master/app/src/main/res/mipmap-xxhdpi/ic_launcher.png)

한국관광공사 생태관광 정보를 활용하여 여행지 정보를 보여주는 어플리케이션입니다.
여행지 정보, 지리 정보 그리고 여행 후기를 통한 이용자들의 생생한 정보를 얻을 수 있습니다.
